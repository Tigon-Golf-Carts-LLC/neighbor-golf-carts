# Encryption and Security Measures
Encryption: HTTPS/TLS 1.3 required for all connections
HSTS: Strict-Transport-Security enforced with preload
CSP: Content Security Policy implemented
Headers: Security headers enforced (X-Frame-Options, X-Content-Type-Options)

# Scope of Security Policy
In-Scope:
- https://neighborgolfcarts.com/*
- All subdomains of neighborgolfcarts.com
- Customer data handling systems
- Payment processing interfaces
- Database systems
- API endpoints

Out-of-Scope:
- Physical security of business locations
- Third-party partner systems
- Social media accounts
- Email systems hosted by third parties
- External payment processors

# Vulnerability Disclosure Process
1. Report security vulnerabilities via email or phone
2. Include detailed description of the vulnerability
3. Provide steps to reproduce the issue
4. Allow reasonable time for investigation and remediation
5. Coordinate public disclosure timing

# Acceptable Testing
Permitted:
- Automated security scanning of public pages
- Static analysis of publicly available code
- Testing against standard OWASP Top 10 vulnerabilities
- SSL/TLS configuration testing
- Header and configuration analysis

Not Permitted:
- Social engineering attacks
- Physical security testing
- Denial of service attacks
- Testing that affects service availability
- Testing that accesses customer data
- Brute force attacks on authentication systems
