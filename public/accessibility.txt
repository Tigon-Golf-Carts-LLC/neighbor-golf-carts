# Accessibility Statement - Neighborhood Golf Carts
# Commitment to Digital Accessibility and Inclusive Design

## ACCESSIBILITY COMMITMENT
Neighborhood Golf Carts is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards to ensure our website is accessible to all users nationwide.

## ACCESSIBILITY STANDARDS COMPLIANCE
Our website aims to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards. These guidelines explain how to make web content more accessible for people with disabilities and user-friendly for everyone.

### Current Compliance Level
- **Target Standard:** WCAG 2.1 AA
- **Testing Method:** Automated and manual accessibility testing
- **Last Audit:** November 24, 2025
- **Next Review:** Quarterly accessibility assessment

## ACCESSIBILITY FEATURES IMPLEMENTED

### Keyboard Navigation
✓ Full keyboard navigation support
✓ Visible focus indicators
✓ Logical tab order throughout the site
✓ Skip links for main content navigation
✓ Keyboard shortcuts for common actions

### Screen Reader Support
✓ Semantic HTML structure for proper screen reader interpretation
✓ ARIA labels and descriptions where appropriate
✓ Alternative text for all meaningful images
✓ Descriptive link text and button labels
✓ Proper heading structure (H1-H6 hierarchy)

### Visual Design Accessibility
✓ High contrast color combinations
✓ Color-blind friendly color palette
✓ Scalable text up to 200% without horizontal scrolling
✓ Clear visual focus indicators
✓ Consistent navigation and layout structure

### Content Accessibility
✓ Plain language usage where possible
✓ Clear and descriptive page titles
✓ Logical content structure and organization
✓ Error messages are clear and helpful
✓ Forms include proper labeling and instructions

### Interactive Elements
✓ Form fields with associated labels
✓ Clear error messaging and validation
✓ Accessible dropdown menus and navigation
✓ Proper button and link identification
✓ Modal dialogs with appropriate focus management
